## Technologies

Project is created with:

- [Apollo-server.js](http://expressjs.com/) node.js web application framework for server
- [Express.js](http://expressjs.com/) node.js web application framework for server
- [GraphQL](https://graphql.org/) query language for APIs
- [express-graphql](https://graphql.org/graphql-js/express-graphql/) creates Express server that runs a GraphQL API.
- [MongoDB](https://graphql.org/) classified as a NoSQL database program, MongoDB stores data in JSON-like documents.
- [Mongoose](https://mongoosejs.com/) mongoose provides a straight-forward, schema-based solution to model your application data.

- [Nodemailer](https://mongoosejs.com/) mongoose provides a straight-forward, schema-based solution to model your application data.
- [Redis](https://mongoosejs.com/) mongoose provides a straight-forward, schema-based solution to model your application data.
- [Node-cron](https://mongoosejs.com/) mongoose provides a straight-forward, schema-based solution to model your application data.
- [Googleapis](https://mongoosejs.com/) mongoose provides a straight-forward, schema-based solution to model your application data.
